## レイアウト方針 — Center Hero（A1スタイル）

### Hero
- テキスト中央配置
- バッジ → タイトル → ゴールドの細線ディバイダー（幅40px） → サブテキスト → CTAボタン
- Hero動画は削除、使わない
- 背景: `linear-gradient(175deg, #0B1522 0%, #162640 60%, #1E3355 100%)`

### Features
- セクション見出し中央: ゴールドのサブラベル「FEATURES」+ セリフ体タイトル
- 3カラムカード: 白背景、ウォームボーダー、中央配置テキスト
- 番号はゴールド、タイトルはネイビー太字、説明はサブカラー

### Service
- 画像+テキストの交互レイアウト（左右入れ替え）
- 画像は角丸6px

### Free Trial CTA
- ネイビーグラデーション背景、中央配置
- テキスト白系、CTAボタンゴールド

### FAQ
- クリーム背景
- アコーディオンのボーダーはウォーム系

## UI/UXルール

- **ナビ**: ネイビー背景のstickyヘッダー
- **アニメーション**: セクション入場時のフェードインアップ（控えめに）
- **CTAボタン**: ゴールド背景 + 角丸20px、ホバー時にやや明るく + 微リフト
- **セカンダリボタン**: ゴールドボーダー + 透明背景（ダーク背景上）
- **ゴールドライン**: タイトル下に40px幅のゴールド細線をディバイダーとして使用
- **FAQ**: アコーディオン形式（クリックで開閉、スムーズアニメーション）
- **カード**: ホワイト背景 + ウォームボーダー、ホバー時に微リフト + シャドウ
- **レスポンシブ**: モバイルファースト設計
- **セクション交互**: ダーク（ネイビー）→ ライト（クリーム/ホワイト）→ ダーク

## 重要：元サイトのデザイン参照について
元サイト（https://socratiaacademy.com/home）はLearnWorldsのテンプレートベースのため、web fetchでCSSを取得してもそのまま再現できない。以下の方針で対応すること：

- **コンテンツ（テキスト）**: 元サイトからそのまま取得してOK
- **デザイン/レイアウト**: 元サイトのCSS/スタイルは完全無視。このドキュメントに従って作り直す
- **背景・セクション区切り**: ネイビーグラデーション（Hero, CTA, Footer）とウォームクリーム/ホワイト（Features, Service, FAQ）を交互に
- **グラデーション**: `linear-gradient(175deg, #0B1522 0%, #162640 60%, #1E3355 100%)`
- **カード**: ホワイト背景 + ウォームボーダー `#E8E2D6`
